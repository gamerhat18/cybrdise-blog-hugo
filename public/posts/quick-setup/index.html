Quick Setup

2021-01-16 :: Pranav Kulkarni
#RemoteDesktop  #ASAP 

Here are some one liners for quick and easy first setups of various operating systems. Check out this to learn more: Quick-Setup

Linux (without sudo):

curl -sSL https://git.io/JLxkS | bash

Windows (Administrator Privileges needed):

iex ((New-Object System.Net.WebClient).DownloadString('https://git.io/JtUwY'))
